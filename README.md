# Sounds of Seattle Birds: Classifying Bird Calls with Neural Networks
**DATA 5322 - Statistical Machine Learning 2 | Seattle University | Spring 2026**
**Author:** Sravya Sri Murala

---

## Overview
This project trains convolutional neural networks (CNNs) to classify bird species 
from mel-spectrogram representations of their calls. The dataset contains 1,981 
spectrograms across 12 Seattle bird species.

## Models
- **Binary CNN** — 3 conv blocks (16/32/64 filters) + GlobalAveragePooling + Dense(32) — 25,409 parameters
- **Multi-class CNN** — 4 conv blocks (32/64/128/128 filters) + GlobalAveragePooling + Dense(128) — 258,316 parameters

---

## Environment
- Python 3.11
- TensorFlow 2.16.2 (Apple Silicon Metal GPU backend)
