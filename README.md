## Overview
This project trains convolutional neural networks (CNNs) to classify bird species 
from mel-spectrogram representations of their calls. The dataset contains 1,981 
spectrograms across 12 Seattle bird species.

## Models
- **Binary CNN** — 3 conv blocks (16/32/64 filters) + GlobalAveragePooling + Dense(32) — 25,409 parameters
- **Multi-class CNN** — 4 conv blocks (32/64/128/128 filters) + GlobalAveragePooling + Dense(128) — 258,316 parameters

---

## Hyperparameter Tuning Summary

The following hyperparameters were systematically varied across three multi-class experiments:

- **Architecture depth:** 3 vs 4 convolutional blocks
- **Filter sizes:** 16/32/64 vs 32/64/128/128
- **Learning rate:** 0.001 vs 0.0005
- **Class imbalance handling:** class weights vs balanced batch sampling vs neither
- **Model capacity:** ~82K vs ~258K parameters
- **Dropout rate:** 0.4 vs 0.5

EarlyStopping patience was tuned in the range 10-12 epochs, and ReduceLROnPlateau used factor 0.5 and patience 5 after evaluating alternatives.

| Experiment | Architecture | LR | Class Handling | Parameters | Best Val Acc | Outcome |
|---|---|---|---|---|---|---|
| v1 (Baseline) | 3 blocks, 16/32/64 | 0.001 | Class weights | 82K | 32.2% | Collapsed to majority |
| v2 (Balanced) | 3 blocks, 16/32/64 | 0.001 | Balanced sampling | 82K | 14.4% | Over-corrected |
| v3 (Final) | 4 blocks, 32/64/128/128 | 0.0005 | None | 258K | 47.3% | Working model |
