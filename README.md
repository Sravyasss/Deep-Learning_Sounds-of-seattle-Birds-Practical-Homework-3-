# Sounds of Seattle Birds — DATA 5322 Practical Homework 3

Convolutional neural networks for classifying bird calls from mel-spectrograms across twelve Seattle bird species.

## Contents
- `bird_classification.ipynb` — Main notebook (binary + multi-class CNNs + mystery clip predictions)
- `figures/` — All plots and figures generated by the notebook
- `select_birds.ipynb` — Preprocessing notebook (reference)
- `data/` — Sample data files (HDF5 spectrograms not included due to size)

## Results Summary
- **Binary classification (House Sparrow vs Spotted Towhee):** 78.4% test accuracy, AUC 0.694
- **Multi-class classification (12 species):** 50.0% top-1 accuracy, 72.8% top-3 accuracy
- **Mystery clip predictions:** Identified Red-winged Blackbird in test2.mp3 (single bird, 78% confidence). Detected multiple species in test1.mp3 and test3.mp3.

## Environment
- Python 3.11, TensorFlow 2.16.2 (Apple Silicon Metal GPU backend)
- See notebook imports for dependencies (h5py, librosa, scikit-learn, etc.)
